## 진행방식
> 각자 자신의 브랜치를 생성해서 매일 푼 문제의 리뷰를 업로드 합니다.
1. 하루에 과제를 파이썬으로 풉니다.
2. 못 풀어도 좋으니 최대한 정답을 보지 않고 시도합니다.
3. 풀고나면 각자 브랜치에 자신의 py코드를 올립니다.
4. 시간을 정해 모두 모여 각자의 코드를 실시간 리뷰합니다.
